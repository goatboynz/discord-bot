Commands

!build_server

Create a complete server structure from your description

Example: !build_server Create a gaming community server
!add channels

Add new channels to your server

Example: !add channels Create support channels
!add roles

Create new roles with permissions

Example: !add roles Create moderator roles
!add content

Add formatted content to channels

Example: !add content Create server rules

Installation

Prerequisites

  • Python 3.8 or higher
  • pip package manager

Install Dependencies

pip install discord.py

Clone Repository

git clone https://github.com/yourusername/discord-server-builder.git

Run the Bot

python bot.py

  • 24/7 running computer

Server Setup

# Run with PM2
npm install pm2 -g
pm2 start bot.py --name discord-bot
Visit DigitalOcean

Docker Deployment

# Dockerfile
FROM python:3.10-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .
CMD ["python", "bot.py"]

# Build and run
docker build -t discord-bot .
docker run -d --env-file .env discord-bot

Environment Variables

Required Variables

  • DISCORD_TOKEN
  • GEMINI_API_KEY

Optional Variables

  • LOG_LEVEL=INFO
  • COMMAND_PREFIX=!

Monitoring

# Monitor with PM2
